# SQL Retail Practice Project

## Project Goal
Remember relational database systems and SQL:
- DDL (schema definition)
- DML (data manipulation)
- TCL (transaction control)
- DCL (data control)
- Complex JOIN queries
- Aggregations

## Database Model
customers → orders → order_items → products

Simulation of a retail relational database.

## Concepts Covered
- Primary Keys
- Foreign Keys
- Referential Integrity
- Transactions (BEGIN, ROLLBACK)
- Access control (GRANT)
- Analytical queries (GROUP BY, SUM, JOIN)
